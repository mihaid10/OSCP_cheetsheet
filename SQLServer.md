+ DB接続ソースなどから認証情報を取得できる場合がある

  ```
  set cnn = server.createobject("ADODB.Connection")
  cnn.open "PROVIDER=SQLOLEDB;DATA SOURCE=RALPH;User ID=sa;PWD=poiuytrewq;DATABASE=bankdb"
  ```

  ※saが管理者ユーザであることが多い
